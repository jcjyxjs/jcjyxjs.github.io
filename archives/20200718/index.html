2020-07-18 jcjyxjs

今天看微信公众号的时候，突然看到这么一条东西。于是我就点进去看了。

大致意思就是： GitHub 在 2020 年 2 月 2 日，对每个活跃的公共存储库进行快照捕获，并保存在 GitHub Arctic Code Vault 中。

并且仓库被快照获取的用户，会给一个 GitHub Arctic Code Vault 奖章，就像这样。

然后这些被快照获取的仓库源代码，会被永久保存到 the Arctic（北极圈） 。

具体可以查看官方介绍 https://archiveprogram.github.com/
